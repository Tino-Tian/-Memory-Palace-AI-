# /宫殿 保存现场

保存当前工作现场的完整快照，下次启动时自动恢复。

**执行步骤：**

1. 生成 brief 摘要（200 字内）写出到 `~/.claude/projects/-Users-mac/memory/session-resume-brief.md`
2. 生成 full 详细记录写出到 `~/.claude/projects/-Users-mac/memory/session-resume-full.md`（含：当前任务进度、未完成项、关键文件列表、上下文关键信息）
3. 调用保存脚本写入数据库：
```bash
SESSION_ID="session-$(date +%Y%m%d-%H%M%S)"
bash ~/memory-palace/hooks/save-session.sh "$SESSION_ID"
```
