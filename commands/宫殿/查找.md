# /宫殿 查找 <关键词>

在记忆宫殿中搜索历史记忆。先查主库，再查归档。

**执行步骤：**

1. 先搜主库（FTS5 全文搜索，支持多词和前缀）：
```bash
sqlite3 ~/memory-palace/palace.db "
SELECT c.path, e.date, e.content FROM entries e
JOIN categories c ON e.category_id = c.id
WHERE e.id IN (SELECT rowid FROM entries_fts WHERE entries_fts MATCH '$1')
ORDER BY e.date DESC LIMIT 10;
"
```

2. 如果主库没找到，搜索归档（归档无 FTS 索引，用 LIKE）：
```bash
for f in ~/memory-palace/归档/*.db.gz; do
  gunzip -kf "$f" && sqlite3 "${f%.gz}" "SELECT date, content FROM entries WHERE keywords LIKE '%$1%' OR content LIKE '%$1%';" && echo "（来自归档: $f）"
done
```

3. 把查到的结果用一两句话告诉用户。
